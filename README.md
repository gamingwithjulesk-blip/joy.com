# joy.com

joy.com is a interactive website full of whimsy and fun centered around being a safe, happy corner of the internet.  
joy.com has all sorts of fetures such as: 
daily affrimations, 
song recemendation genertor, 
a randomly generated color pallet for outfit inspo, 
a gallery where you can upload photos of art/outfits you put together,
a generator for screen free actctivitys,
an area where you can write journal entries, lists, idea, ect and save them in a folder on the website. 
and the most whimsical moth themed interface. 

 i did use AI for this project. i used copiolit for debugging and code suggestions, and i used figgma to help launch my website by putting all the pieces (code, images, and design) and getting it to run it.
