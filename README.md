# joy.com

joy.com is a interactive website full of whimsy and fun centered around the idea that the internet should be a place of connection not discord. 
joy.com has all sorts of fetures such as: 
daily affrimations, 
song recemendation genertor, 
a randomly generated color pallet for outfit inspo, 
a gallery where you can upload photos of art/outfits you make,
aborable free coloring sheets designed by me,
a generater for screen free actctivitys
and the most whimsical interface.

my main insparation for this project was seeing the massive decline in comunity on the internet, a place built to foster comunity which got me thinking 'what if i built a website built to bring people togheter and spark just a little bit of joy?' which summoned the idea from the depths of my brain to make joy.com a website that is specificly designed to spark
joy, creativity, and comunity. 

i used a little bit of AI for this project. i used copiolit for debugging and code suggestions, and i used figgma to help launch my website by putting all the pieces (code, images, and design) and getting it to run it. 
i had no other help for this project and did everything else on my own. 

i hope you enjoy my project!
